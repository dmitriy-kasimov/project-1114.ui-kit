## Link

Пример использования:

```typescript jsx
import { Text, Link } from '@tr271v0r/ui-tool-kit'


export function App(){
    return (
        <>
            <Link>
                <Text>Some link</Text>
            </Link>
            <Link disabled>
                <Text>Some disabled link</Text>
            </Link>
        </>
       
    )
}

```
